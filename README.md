# 🎸 Hitori-Days (ひとりデイズ)
> **A cozy walking simulator RPG fan game inspired by *Bocchi the Rock!***
> *Created by Rei Taylor*

[![GameMaker](https://img.shields.io/badge/Engine-GameMaker-green.svg)](https://gamemaker.io)
[![Status](https://img.shields.io/badge/Status-In--Development-orange.svg)](#)

Dive into a 16x16 pixel-art recreation of Shimokitazawa. Experience the quiet, atmospheric, and occasionally anxious daily life of Hitori "Bocchi" Gotoh in this low-stress RPG. Explore Bocchi's house and areas across Yokohama and Shimokitazawa while interacting with friends.

---

## ✨ Key Features
* **16x16 ROM Aesthetic:** Pure retro visuals optimized for a high-fidelity "handheld" feel with a custom-written NTSC Blargg CRT effect.
* **Living Audio:** 3D Audio Emitters and a dynamic Radio System for immersive soundscapes.
* **Narrative Systems:** Engage with the cast using our custom Follower, Dialogue, and Cutscene systems.
* **Robust Text Rendering:** Powered by Scribble for high-quality, expressive dialogue rendering.

## 🛠️ Technical Stack
* **Engine:** [GameMaker](https://gamemaker.io)
* **Text Rendering:** [Scribble](https://github.com/JujuAdams/Scribble) by JujuAdams
* **Graphics:** Hand-written Blargg NTSC CRT Shader
* **Stability:** Custom C++ DLL for robust unhandled exception logging and crash management
* **Logic:** A hybrid of Custom C++ and GML (GameMaker Language)
